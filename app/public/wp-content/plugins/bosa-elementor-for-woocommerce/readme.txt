== Installation ==

= 1) Install =
Bosa Elementor Addons and Templates for WooCommerce plugin can be installed by two methods.

= Direct Install =

1. Go to the WordPress Dashboard "Add New Plugin" section.
2. Search For "Bosa Elementor Addons and Templates for WooCommerce".
3. Install, then Activate it.

= Zip Install =

1. Unzip (if zipped) and Upload `bosa-elementor-for-woocommerce` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

= 2) Import =

1. After you install and activate the plugin you will get a notice to install Elementor/WooCommerce Plugin ( If already installed then notice is not shown. ).
2. To install the plugin click on the "Activate" Install Elementor/WooCommerce.
3. Now go to WooCommerce - BEW Templates Library to import the Elementor templates.

= 3) Elementor Widgets =

All our WooCommerce Elementor Widgets are available in Elementor builder under "BEW ELEMENTS".

== Frequently Asked Questions ==
= Does it work with any WordPress theme? =

Yes, it will work with any WordPress theme as long as you are using WooCommerce and Elementor as a page builder. However, some dynamic elements, such as a product or product category, appear in a different layout that is defined by the theme. Some themes override WooCommerce product styling and this can make some widget options inactive.

= My WooCommerce products look different as in the demo. =

The products or categories styling is loaded from the theme enabled on your website. Each theme that supports WooCommerce can have own products styling.

= Is the Elementor PRO required? =

No, Elementor PRO or any other PRO addon is not required.
